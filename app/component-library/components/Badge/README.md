# Badge

Badge is a component used to display text within a container.

## Props

This component extends `ViewProps` from React Native's [View Component](https://reactnative.dev/docs/view).

### `label`

Label within the Badge.

| :-------------------------------------------------- | :------------------------------------------------------ |
